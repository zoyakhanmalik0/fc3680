# Other Game Logos

Put your other game logo images in this folder.

The current page uses these filenames:

- `3patti-crown.svg`
- `3patti-room.svg`
- `lucky-33.svg`
- `3patti-pearl.svg`
- `3patti-no1.svg`
- `3patti-ok.svg`
- `teenpatti-show.svg`
- `3patti-free.svg`
- `3patti-blue.svg`

You can replace them with `.png`, `.jpg`, or `.webp` files. If the extension changes, update the matching `src` in `index.html`.
